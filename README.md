# bi-cycle
A flutter demo application to demonstrate:
* ```BLoC``` state management
* Fetch get request 
* ```JSON``` data manipulation
* Classes ```Models``` construction

# How to run
```flutter pub get``` then ```flutter pub run``` and it's done.
